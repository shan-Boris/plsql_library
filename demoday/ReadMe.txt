Выполнено небольшое тестовое покрытие функций и процедур с помощью sh скриптов, 
делая запросы с помощью node js. для запуска тестов необходимо:
(Установленная node)(находимся в test/)
1. npm init -y
2. npm install oracledb
3. иметь пользователя в базе логином/паролем как в js файла
(либо изменить параметры подключения в js файлах)
4. запустить ./test.sh
